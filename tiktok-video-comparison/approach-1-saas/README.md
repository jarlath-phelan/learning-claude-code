# Approach 1: SaaS Tools (Full AI Generation)

Use existing AI video platforms with optimized prompts.

## Recommended Tools

| Tool | Best For | Cost | Time to Video |
|------|----------|------|---------------|
| InVideo AI | Quick TikTok clips | Free-$25/mo | ~5 minutes |
| Synthesia | Professional avatars | ~$22/mo | ~10 minutes |
| Canva AI | Simple animations | Free | ~3 minutes |
| Revid.ai | Viral optimization | Varies | ~5 minutes |

## Usage

Copy the prompts from `prompts/` directory into your chosen tool.

## Files

- `prompts/uno-no-mercy.txt` - UNO No Mercy explainer prompt
- `prompts/financial-basics.txt` - Financial basics explainer prompt
- `prompts/generic-template.txt` - Template for any topic
